
# **Pokédle**

Everyone's favorite daily game, but Pokémon themed!

Pokédle is a browser based game where you have 6 tries to guess the 5-letter Pokémon of the day. 

## How to play

**Guess the Pokémon in 6 tries**

Start typing your first guess, hit enter and watch the color of the tiles change!

After each guess, the tiles' color will indicate how close you were to the answer:

- Green: The letter is in the Pokémon's name and in the correct position
- Yellow: The letter is in the Pokémon's name but in the incorrect position
- Grey: The letter isn't in the Pokémon's name

## Run Locally

```bash
  git clone https://github.com/rebecca-marusca/Pokedle.git
  cd Pokedle
  npm install
  npm run start
```

## Why?

Pokédle (my second ever web project) is something I've been looking forward to building for a while. 
I love my daily dopamine dose of browser games and decided to put my own spin on a wordle clone, and made Pokédle.

I've loved Pokémon since I was little and my older brother taught me the art of the GBA Emulator on my barely functional Motorola phone, and to this day the obsession hasn't left me, so here we are!

I'm definitely still not in my element in web, but this was a fun way of getting some practice in and showing my love for this gaming franchise.
